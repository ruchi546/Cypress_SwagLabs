# Cypress Swag Labs - Portfolio Project

UI test automation project using Cypress on the Swag Labs (saucedemo) web app.

## Project Overview

I built this suite to validate key customer flows from login to checkout, with a structure that is easy to maintain and scale.

## What This Demonstrates

- End-to-end test automation for user journeys
- Page Object Model for reusable, clean test design
- Test data management with fixtures
- Organized test suites by feature area

## Covered User Flows

- Authentication: valid and invalid login behavior
- Inventory: product visibility and cart actions
- Cart: item review and checkout readiness
- Checkout: form flow and order completion behavior
- Sidebar: navigation and session-related actions

## Tech Stack

- Cypress
- JavaScript
- Node.js and npm
